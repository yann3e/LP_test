# LP_test

Landing Page test project for the Automateez Agency.

## 📌 Project Overview

This repository contains resources, prototypes, and development files used to build and test landing pages for **Automateez Agency**.

The goal of this project is to experiment with:

* Modern landing page layouts
* Conversion-focused UI/UX
* Responsive design
* Performance optimization
* Marketing-oriented web components

---

## 🚀 Technologies

Depending on the iteration, the project may include:

* HTML5
* CSS3
* JavaScript
* Angular
* Bulma CSS
* Responsive Web Design principles

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/LP_test.git
```

Navigate into the project folder:

```bash
cd LP_test
```

Install dependencies if needed:

```bash
npm install
```

Run the development server:

```bash
npm start
```
